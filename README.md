# Challenge-converson-de-moneda
Conversor de Moneda
 Conversor de Monedas - Java Console App
Este proyecto es un desafio propuesto por el programa ALURA ONE.
Es un conversor de monedas interactivo desarrollado en Java, que utiliza la API ExchangeRate-API para obtener tasas de cambio en tiempo real!
La aplicación opera desde la consola y permite a los usuarios realizar conversiones entre pares de divisas predefinidos, además de guardar un historial de todas las transacciones realizadas.
